The directory for AI governance.

We track the files that govern what AI coding agents can do — instruction files, guardrail frameworks, compliance standards, and Agent Skills. Search, filter, and find what applies to your codebase.

What counts as a "governance file"? Anything that tells an agent what it's allowed to do — a CLAUDE.md/AGENTS.md scoping its behavior on your repo, a guardrails framework filtering its outputs, a compliance standard bounding how an org may use it. A Skill is a governance file too — a scoped instruction set that governs how an agent should approach one specific task.
